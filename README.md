# cosmos-psychology
Software Description: An interactive sandbox game where by mixing emotions and psychological concept you learn more about psychology in general. This is all done in a solar system kind of framework.

Motivation: To make diagnostic psychological frameworks (DSM-5 criteria sets) highly interactive and approachable through alchemy-style puzzle mechanics.

Beneficiaries: Psychology students learning diagnostic patterns, educational institutions looking for gamified learning mechanics, and indie game enthusiasts interested in abstract state management and expanding on this concept to take on other psychological concepts.
